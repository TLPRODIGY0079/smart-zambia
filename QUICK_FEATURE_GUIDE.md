## 1. Services Tab

### How to Access
1. Open Smart Zambia app
2. Click "Services" tab in navigation
3. Choose a category

### Categories
- 🍽️ **Restaurants** - 5 dining options
- 🔧 **Repair Services** - 4 repair shops
- 🏠 **Real Estate** - 4 property listings
- 💼 **Professional Services** - 5 service providers

### Actions
- Click service card → View details
- Click "Call Now" → Opens phone dialer
- Click "Book Service" → Send booking request
- Earn XP: +3 view, +5 call, +10 book

## 2. Easter Eggs

### 🎮 Konami Code
**How**: Press ↑↑↓↓←→←→BA on keyboard
**Reward**: +1000 XP + Secret Agent badge
**Effect**: Screen shake + confetti

### 🔍 Logo Click
**How**: Click the "Z" logo 7 times quickly
**Reward**: +500 XP + Curious Explorer badge
**Effect**: Logo spins

### 🦉 Midnight Login
**How**: Login between 00:00-01:00
**Reward**: +200 XP + Night Owl badge + 2x XP for 1 hour
**Effect**: XP multiplier

### 🎰 Lucky Numbers
**How**: Visit site on 7th, 77th, or 777th visit
**Reward**: +100, +500, or +2000 XP
**Effect**: Fireworks

### 🔐 Hidden Codes
**How**: Find "ZAMBIA2024", "VICTORIA", or "COPPER" in descriptions
**Reward**: +250 XP per code
**Effect**: Achievement toast

### ⚡ Speed Demon
**How**: Complete 5 actions within 30 seconds
**Reward**: +300 XP + Speed Demon badge
**Effect**: Achievement unlock

### 🇿🇲 Profile Secret
**How**: Set your bio to include "I love Zambia"
**Reward**: +200 XP + Patriot badge
**Effect**: Achievement unlock

**Total Available**: 4,550+ XP from all Easter eggs

## 🔧 Troubleshooting

### Services Not Loading
- Refresh the page
- Clear browser cache
- Check internet connection

### Easter Egg Not Unlocking
- Make sure you haven't already unlocked it
- Try again (some require exact timing)
- Check console for errors

### Admin Can't Login
- Verify credentials are correct
- Clear browser cache
- Try incognito mode
- Check localStorage is enabled

### Data Not Saving
- Check localStorage is enabled
- Don't use private/incognito mode
- Clear cache and try again
